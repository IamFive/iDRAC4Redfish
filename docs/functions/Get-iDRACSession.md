# Get-iDRACSessions

Get-iDRACSession connect´s to an iDRAC Controller ussing Session based Authentication

```Powershell
NAME
    Get-iDRACSessions

SYNTAX
    Get-iDRACSessions 
```

Lists all iDRAC Sessions currently open
```Powershell
Get-iDRACSessions
```
